# sites-lejing-guide

## 项目简介

`sites-lejing-guide` 是一个用于归档和发布多个独立 HTML 页面的仓库。本仓库不针对任何单个域名或具体网站，而是作为一个通用的静态页面集合，便于对各类网页内容进行集中管理与版本控制。

## 目录说明

```
sites-lejing-guide/
├── docs/          # 主要归档目录，存放各 HTML 页面及相关资源
├── assets/        # 静态资源文件（图片、样式、脚本等）
├── README.md      # 本文件
└── .gitignore     # Git 忽略规则
```

- `docs/`：存储所有归档的 HTML 页面及其子目录，每个页面可按功能或主题独立组织。
- `assets/`：存放页面共用的资源文件，如 CSS、JavaScript 和图片等。

## 页面归档说明

本仓库中的 HTML 页面均为独立文件，可直接通过浏览器打开查看。页面内容涵盖多种类型，适用于学习、参考或快速展示。

- 每个页面保持自包含，尽量减少对外部资源的依赖。
- 资源引用路径统一使用相对路径，便于本地预览和部署。
- 如需添加新页面，请将其放入 `docs/` 目录下，并保持文件命名清晰。

## 维护说明

- 本仓库由维护者不定期更新，添加或修改已有页面。
- 欢迎提交 Issue 或 Pull Request 以提出改进建议或报告问题。
- 所有页面内容应遵守相关法律法规，不得包含不当信息。
- 修改后请确保页面在主流浏览器中正常显示。
